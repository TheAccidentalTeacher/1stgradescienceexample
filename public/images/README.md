# Images Directory

Place lesson images in organized subfolders:

## Structure:
```
images/
  unit1/
    day1-light.jpg
    day2-sky.jpg
    day3-land.jpg
    ...
  unit2/
    trex.jpg
    triceratops.jpg
    ...
  unit3/
    excavator.jpg
    bulldozer.jpg
    ...
```

## Image Guidelines:
- Large, clear images (at least 800x600px)
- Simple, not cluttered
- Bright colors
- Kid-friendly content
- Real photos or clear illustrations

## AI Image Generation Prompts:

### For Educational Content:
- "Simple, colorful illustration of [topic] for a 7-year-old, educational style"
- "Photorealistic image of [subject], bright lighting, clear details"
- "Cartoon-style drawing of [concept], vibrant colors, kid-friendly"

### For Biblical Scenes:
- "Biblical creation scene, [specific day], reverent artistic style"
- "Noah's ark illustration, colorful, suitable for children"

Remember: Images should support Bobby's visual learning style and accommodate his reading level by reducing text dependence.
